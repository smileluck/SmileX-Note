# pip 安装和配置

> 创建conda环境会默认安装pip

 **若没有安装conda，需要安装python 解释器，python新版本的都自带pip，无需另外安装。** 

1. 配置清华源

   在 `C:/users/[用户]` 文件夹下，创建文件夹 `pip` ，再在其目录下创建 `pip.ini` 文件，并输入如下内容。

   ```ini
   [global]
   index-url = https://pypi.tuna.tsinghua.edu.cn/simple
   [install]
   trusted-host = pypi.tuna.tsinghua.edu.cn
   ```
