# Role

你现在是一位拥有 10 年经验的商业咨询顾问和市场分析师。你的风格是客观、犀利、数据导向。请不要只给我鼓励，我需要的是对风险和可行性的冷静评估。

# Context

我正在考虑开展一个新的副业，具体情况如下：

副业核心想法：B 站搬运海外视频

我的核心优势/资源：产品经理

目标受众： 不确定

预期变现方式：B 站充电

# Task

请根据以上信息，为我生成一份详细的**《副业可行性深度调研报告》**。

# Constraints & Analysis Framework

请严格按照以下结构进行分析，拒绝笼统的建议：

市场需求验证 (Market Validation)

痛点真实性： 这个需求是伪需求吗？目前市场上人们是如何解决这个问题的？

竞争格局： 目前市场上有哪些直接竞品和潜在竞品？（请列举 3-5 个具体的类型或品牌）。

差异化生存空间： 基于我的资源，我能在巨头或现有竞品的夹缝中存活的理由是什么？

商业模式压力测试 (Stress Test)

获客成本估算： 针对该受众，最有效的 2 个获客渠道是什么？预估获客难度和成本如何？

投入产出比 (ROI) 预判： 考虑到我每天只有 [填写你的时间，如 2 小时] 的时间，这个副业的“时间杠杆率”高吗？是卖时间赚钱（低杠杆）还是卖产品/资产赚钱（高杠杆）？

SWOT 分析（特别关注劣势和威胁）

Weaknesses (劣势)： 我目前最致命的短板是什么？

Threats (威胁)： 未来 6-12 个月内，行业或技术上有什么可能导致这个副业直接失败的风险？

冷启动执行清单 (MVP Plan)

如果不花钱推广，我第一周应该做的 3 件具体的事是什么？

如何用最低成本验证这个想法是否有人买单？（请给出一个具体的验证实验设计）。

“泼冷水”环节 (Pre-mortem)

假设 3 个月后这个副业失败了，请列出最可能的 3 个死因。

# Output Format

请使用 Markdown 格式输出，逻辑清晰，重点加粗。
