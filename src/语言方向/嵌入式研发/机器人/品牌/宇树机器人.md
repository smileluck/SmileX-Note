1. app无法配置sta模式，会卡在第一步连接机器人
```shell
# 通过命令行控制连接
sudo ifconfig wlan0 up
sudo nmcli radio wifi on
nmcli radio wifi
nmcli device wifi connect <SSID> password <password>
```
   1. wifi 名称需要最好只有字母
   2. wifi 密码支持英文+数字，最好8位

2. app无法显示摄像头视频
   1. 摄像头固件需要升级
   2. https://support.unitree.com/home/zh/G1_developer/FAQ#heading-17
