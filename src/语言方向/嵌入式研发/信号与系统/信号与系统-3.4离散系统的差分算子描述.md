# 差分算子E的定义

- 延迟算子$E^{-1} $
  
  - $E^{-1}=f(k-1)$
  
  - $E^{-2}=f(k-2)$
  
  - $E^{-n}=f(k-n)$

- 超前算子$E$
  
  - $E^{1}=f(k+1)$
  
  - $E^{2}=f(k+2)$
  
  - $E^{n}=f(k+n)$
