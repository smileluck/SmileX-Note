# 行业周期研究

## 经济周期

### 经济周期是什么
经济周期是无数人类个体和组织在应对经济关系过程中为了生存、繁衍和发展，在自然周期、生命系统周期、人类情绪周期、社会周期等周期规律影响下做出的名种经济活动的总和。
这个总和形成的合力会在内外各种因素和力量影响下，在运动方向、波动强弱、持续时间长短、影响范围大小上不断发生变化，会外在表现出一定周期现象和规律性，这就是人类社会经济系统中各种周期现象和周期规律出现的本质来源。
这个合力在波动强弱上有强弱之分，因此就有了强周期行业和弱周期行业之分。

### 强周期行业和弱周期行业
> 高负债会带来高经营杠杆和财务杠杆，放大业绩盈亏影响，这对任何行业和企业都是普遍适用的。突发事件如自然灾害、重大事故、疫情、战争等可能会在短期造成行业供需的严重失衡。因此按照双盲对照试验思维，我们要先**剔除高负债、突发事件这两个影响因素再去比较行业周期强弱影响因素**。

- 强周期行业定义:行业产品销售价格或生产成本周期波动范围较大，致使销售价格经常较大幅度偏离生产成本，进而导致行业利润周期波动较大的行业。
- 弱周期行业定义:行业产品销售价格或生产成本周期波动范围较小，销售价格与生产成本之间价差较为稳定，进而导致行业利润周期波动较小的行业。

### 强弱周期的形成因素

1. 产品销售价格市场化定价程度
    - 如果某个行业的产品销售价格是高度市场化定价，不会受到政府或行业协会限制那供需矛盾因子占比就会更高，价格波动会更大。
2. 市场化定价偏离成本价程度、
   - 当产品销售市场化价格大幅波动，经常性大幅偏离成本价时，往往会刺激企业短期的大幅囤货或去库行为，以及中长期的大幅扩大或缩小产能行为，进而形成行业短期的库存周期和中长期的投资周期。
   - 值得注意的是，企业产品单位成本价不是一成不变的，不同的行业成本构成不一致，当燃料、材料等可变成本占比较高且高度市场化定价时，单位成本也会大幅波动，从而导致销售价格大幅偏离成本价。
3. 供给侧扩张产能壁垒
   - 供给侧壁垒大体可分为政策准入壁垒、自然资源壁垒、专利壁垒、技术工艺壁垒,品牌壁垒、管理壁垒等，这些壁垒的共同特征就是限制新的竞争者进入或提高新竞争者的进入成本。
   - 这些壁垒对不同行业的周期性影响应不同阶段具体分析。但有一个共同特征是，当某个行业拥有以上越多的壁垒，就越能限制新竞争者进入，从而使这个行业呈现比较明显的弱周期特征。反之，壁垒越少行业进入门槛就越低，容易吸引大量新竞争者进入或退出，从而呈现更强的周期波动。
   - 当某个行业同时具备产品价格市场化定价程度高、经常性大幅偏离生产成本、供给侧壁垒较低三个特征时，那这个行业往往就会成为强周期性行业。
4. 库存周期对上中下游行业的差异化影响总结
   - 本文核心围绕库存周期的形成逻辑、对上中下游的差异化影响、周期闭环及周期结束的关键判断因素展开，结合市场竞争中的猜疑链与囚徒困境，解释了库存周期的运行规律及行业影响特征。
   - **核心底层逻辑：猜疑链与囚徒困境**
     - 经济活动的核心目标是生存、发展，市场激烈竞争为常态，企业难以判断竞争对手的真实想法与实力，会优先采取对自身最有利的策略保障生存，且默认竞争对手也会如此，进而形成“先下手为强”的行为模式，这一逻辑可解释资源抢夺、企业资金链断裂、金融危机等诸多经济现象。
   - **库存周期的形成与行业差异化影响**
     - 当全球基础资源供给不足且形成**一致性短缺预期**时，猜疑链与囚徒困境会触发各国、各行业、企业及家庭的囤货储备行为，库存周期由此启动，且对行业的影响呈现**上游强、中下游弱**的特征，上游行业周期波动强度显著更大。
     - 规模与库存基数差异：全球中游制造业规模是上游基础原材料采掘业的4-5倍，而中下游各环节的资源库存储备是上游采掘业的10倍左右（有色金属尤为明显）；
     - 囤货引发的供需错配：中下游仅增加1个月的订单库存储备，对应上游矿业10个月的产量，而上游资源短期扩产难度大，无法满足激增的订单需求，直接引发上游产品大幅涨价；
     - 恶性循环加剧短缺：上游涨价进一步强化中下游的囤货、抢货意愿（兼顾生产保障与库存增值），上游企业也会加入惜货、囤货行列，最终导致上游资源短缺情况持续恶化。

   - **库存周期的闭环结束过程**
     - 库存周期的涨价阶段会形成连锁反应，最终触发供需格局反转，完成周期循环，整体路径为：
     - 上游涨价→下游产品被迫提价，消费需求受抑制、销量下滑→中游企业成本高企+需求不足，陷入亏损并被迫去产能、去库存→上游在暴利驱动下大规模扩产→上游资源产能过剩，价格开始下降→产能过剩预期形成，中下游抛售大量库存（抛售1个月库存对应上游10个月销量）→上游库存积压、销售困难，价格暴跌，库存周期完成闭环。

   - **判断全球资源短缺及库存周期结束的三大关键因素**
     - 本轮全球资源短缺是否终结，核心取决于三大因素是否发生**根本性改变**：上游是否实现大规模增产；下游消费需求是否出现大规模减少；中游制造业企业是否大量破产并完成去产能。
